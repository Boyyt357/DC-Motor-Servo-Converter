1. Libraries to Install (via Library Manager)
In your Arduino IDE, go to Sketch -> Include Library -> Manage Libraries... and search for:

ESPAsyncWebServer: Search for "ESPAsyncWebServer" by me-no-dev.

Note: This library often requires a "dependency" library called AsyncTCP (also by me-no-dev) to function. Make sure to install both.

AsyncTCP: Search for "AsyncTCP" and install it.
