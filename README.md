1. Libraries to Install (via Library Manager)
In your Arduino IDE, go to Sketch -> Include Library -> Manage Libraries... and search for:

ESPAsyncWebServer: Search for "ESPAsyncWebServer" 

Note: This library often requires a "dependency" library called AsyncTCP to function. Make sure to install both.

AsyncTCP: Search for "AsyncTCP" and install it.
